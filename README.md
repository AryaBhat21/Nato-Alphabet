# NATO Alphabet Converter

A simple Python program that converts a user-entered word into the NATO phonetic alphabet.

## Files

- `main.py` - reads the NATO alphabet CSV and converts input words into phonetic code words.
- `nato_phonetic_alphabet.csv` - contains the NATO alphabet mapping from letters to code words.

## Requirements

- Python 3.x
- `pandas` library

## Install Dependencies

```bash
pip install pandas
```

## Run

From the project folder, run:

```bash
python main.py
```

Then enter a word when prompted.

## Example

Input:

```
Enter a word: hello
```

Output:

```
['Hotel', 'Echo', 'Lima', 'Lima', 'Oscar']
```
